How to configure your Jabber client?

  • Create a Jabber account guest@<host> where host is the machine on which you're running the server.
  • The password can be any desired password but must not be empty.
  • Your Jabber client should use port 5222 (no SSL).
  • Use the form below to send an invitation to your newly created account.
